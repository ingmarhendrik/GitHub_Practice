Object Diagram

Objektiskeem ehk Object Diagram näitab modelleeritud süsteemi struktuuri täielikku või osalist vaadet konkreetsel ajal. Objekti diagramm keskendub objektide komplekti atribuutidele ja sellele, kuidas need objektid on üksteisega seotud.

Objektdiagrammide kasutamine on üsna piiratud, peamiselt andmestruktuuride näidete näitamiseks. Objektdiagrammid esindavad klassidiagrammi eksemplari. Klassiskeemide ja objektidiagrammide põhimõisted on sarnased. Objektdiagrammid esindavad ka süsteemi staatilist vaadet, kuid see staatiline vaade on süsteemi hetketõmmis konkreetsel hetkel.
